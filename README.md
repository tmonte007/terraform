## Usage
Create<br />
1.) Place a *.tf file in the directory<br />
2.) terraform plan<br />
3.) Terraform apply<br />

Verify<br />
terraform show<br />
or<br />
terraform graph<br />

Destroy<br />
terraform destroy<br />
